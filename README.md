# Gerador de Assinatura de E-mail

## Como abrir no navegador

Você pode abrir o arquivo `index.html` diretamente no navegador ou iniciar um servidor local simples.

### Opção 1 — abrir o arquivo diretamente

1. Abra o arquivo `index.html` com o navegador de sua preferência.

### Opção 2 — servidor local (recomendado)

1. No terminal, execute:

```bash
python -m http.server 8000
```

2. Em seguida, acesse no navegador:

```
http://localhost:8000
```
